Separation of Concerns

What the data layer handles vs what the agent handles

The common anti-pattern is letting the agent handle everything: joins, aggregations, business logic, and language. The right pattern splits the work — the data platform encodes all business rules deterministically, and the agent focuses only on language and orchestration.

Modular Knowledge

CLAUDE.md and SKILL.md working together

CLAUDE.md gives the agent global context — project structure, environment config, and SQL execution rules. SKILL.md files add domain-specific expertise per product line, loaded only when the agent detects a relevant question. Together they keep the context window efficient while providing deep coverage.

Why AgentCore for production analytics agents
